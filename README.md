# zeeblibri

CLI tool and Go module for seaching and download books from Z-Library


## Usage:
```
./zeelibri harry potter 
1. Harry Potter 1
2. Harry Potter 2
3. Harry Potter 3
4. Harry Potter 4
5. Harry Potter 5

> 1

Downloading...
Title: Harry Potter 1
Author: JK Rawling
Size: 129KB EPUB

```


### Note:

```
Don't use this tool for piracy. Please buy books from official sources and support authors.
```
